backgroung color - color de fondo

Si no se espefifica un color de fondo, el fondo se transparente.

CSS trata cada elemento HTML como un cuadro y la propiedad backgroung-color establece de color del fondo para el cuadro. se puede especificar un color de fondo de la mismas tres formas en que puede especificar colores .tambien hemos utilizado la propiedad padding para separar el texto de los bordes de los cuadros . esto hace que sea mas facil de leer.
