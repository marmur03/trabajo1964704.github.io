Letter and Word Spacing

El Kerning es el termino que utilizan los tipografos para el espacio entre cada letra. puede controlar el espacio entre cada letra con la propiedad: letter-spacing. Es particularmente util para aumentar el kerning cuando su encabezado o frase esta en mayusculas. si el texto esta en un caso o normal, aumentar o disminuir el kerning puede dificultar la lectura.

el intervalo predeterminado

Cuando se especifica un valor para estas propiedades, se debe dar en EMS. El intervalo predeterminado entre las palabras se establece por el tipo de letra (a menudo alrededor de 0.25 EM), y es poco probable que deba cambiar esta propiedad con regularidad.
